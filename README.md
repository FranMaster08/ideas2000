# Proyecto Ideas2000

Este proyecto es una aplicación para la gestión de salones y alumnos. A continuación se detallan las características y la estructura del proyecto.

## Estructura del Proyecto

```
package.json
src/
	main.js
	common/
		Isalon.js
	controllers/
		salones.controller.js
	models/
		entity/
			alumno.js
			profesor.js
			salon.js
		services/
			salones.service.js
	views/
		salones.view.js
```

## Instalación

1. Clona el repositorio:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd ideas2000
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```

## Uso

Para iniciar la aplicación, ejecuta:
```bash
node src/main.js
```
